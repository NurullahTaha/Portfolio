# Photography Portfolio

A sleek, professional full-stack photography portfolio built with Next.js 14, Tailwind CSS, and Framer Motion.

## Features
- **Magic Folders:** Interactive 3D hover effects for categories.
- **Admin Dashboard:** Secure upload system to add photos directly from the gallery page.
- **Gear Timeline:** A history of your equipment.
- **Dark Pastel Theme:** Custom aesthetic.

## Getting Started

1. **Install Dependencies** (If you haven't already):
   ```bash
   npm install
   ```

2. **Start the Server**:
   ```bash
   npm run dev
   ```

3. **Visit the Site**:
   Open [http://localhost:3000](http://localhost:3000)

## Admin Access

To upload photos, you must log in.

- **Login Page:** [http://localhost:3000/login](http://localhost:3000/login)
- **Email:** `admin@example.com`
- **Password:** `password123`

Once logged in, go to any **Gallery Category** (e.g., Nature), and you will see the **Upload Zone**.
